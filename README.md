# votacao-personagem
Sistema de Votação de Personagens

Este projeto é um sistema simples de votação onde cada aluno pode votar em um personagem da lista.
Os votos são contabilizados e exibidos em tempo real no ranking da página.

## Funcionalidades
 Votação

O usuário seleciona um personagem através de um menu suspenso.

Ao clicar em Votar, o voto é contabilizado.

Caso nenhum personagem seja selecionado, o sistema exibe um alerta.

 Ranking Atualizado

Mostra quantos votos cada personagem recebeu até o momento.

Lista inclui: Mario, Luigi, Peach, Bowser.

 Informações no Rodapé

Exibe nome do aluno e a data.
